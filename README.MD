# Slots Game

This is a simple game created using SwiftUI and Swift.

## Usage
The game can be played in an iOS simulator. 

### Rules:
- Spin to get new slots.
- If all slots match you earn +30 credits.
- If all slots do not match, you lose -5 credits.
